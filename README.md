PDF Malware Detection
===========

## Usage
```
./main.py <dirname>
```

**Note:** Within the directory, *<dirname>*, PDFs must be labeled as *CLEAN* or
*INFEC* (i.e. "CLEAN_<filename>.pdf" or "INFEC_<filename>.pdf").

## Dependencies
* Python3
* OpenCV
* Progressbar
* skimage
* python3-tk (tkinter)
* sklearn
